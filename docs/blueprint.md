# **App Name**: AfriRep Mobile

## Core Features:

- Phone Verification: Implement phone verification using African country codes for user authentication.
- Reputation Dashboard: Visualize the user's reputation score and display achievement badges.
- Vouching System: Enable users to vouch for each other's skills with confidence ratings and evidence uploads.
- Opportunity Marketplace: Gate access to opportunities based on reputation and filter regionally.
- Financial Hub: Manage multi-currency balances and track payment history without crypto jargon.
- Inner Circles (DAOs): Facilitate transparent governance voting and group savings within communities.
- Regional Adaptation Tool: Tool for automatically adjusting the user interface elements like displaying appropriate language, currency and adapting other elements based on the user's current region, powered by generative AI.

## Style Guidelines:

- Primary color: #008751 (Naija Green) for West Africa; representing growth, freshness, and prosperity.
- Background color: #E0F2F7; a very light tint of the primary, creating a calm background to make other elements pop.
- Accent color: #DC2626; offering a vivid signal to draw the eye.
- Headers: 'Inter' (sans-serif) Bold for headlines.
- Body: 'Inter' (sans-serif) Regular for body text.
- Adinkra symbols (West Africa), Maasai patterns (East Africa), Ndebele art (Southern Africa), Islamic geometrics (North Africa), Kuba cloth patterns (Central Africa)
- Mobile-first design with a focus on accessibility and cultural context.
